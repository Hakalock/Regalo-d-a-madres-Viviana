# Regalo Día de las Madres para Viviana Briñez

Este proyecto tiene tres archivos principales:

- `index.html`: estructura de la página.
- `styles.css`: diseño, colores y animaciones.
- `script.js`: efectos interactivos, pétalos, corazones y sorpresa.

## Cómo abrirlo

1. Descomprime la carpeta.
2. Abre el archivo `index.html` en cualquier navegador.
3. Disfruta el regalo.
